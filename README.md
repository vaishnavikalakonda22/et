# AI Money Mentor 🚀

> An AI-powered personal finance dashboard designed to turn confused savers into confident investors. Built for the ET Gen AI Hackathon.

## 🌟 Overview
95% of Indians don't have a financial plan. AI Money Mentor lives inside the browser as an intelligent financial sidekick, demystifying wealth creation. From diagnosing your 6-dimension financial health to optimizing tax regimes, AI Money Mentor offers an intuitive, gamified, and beautiful user experience. 

## 🛠 Features
- **6-Dimension Financial Health Score**: Instantly calculates your "Money Health Index" across Emergency preparedness, Insurance, Investments, Debt, Taxes, and Retirement.
- **Mutual Fund Portfolio X-Ray (CAS Analyzer)**: AI-driven audit of your CAMS/KFintech Consolidated Account Statements to uncover fund overlap, excessive expense ratios, and underperformance.
- **Tax Optimizer Wizard**: Dynamically models the Old vs. New tax regime given your HRA, 80C, 80D, and Home Loan interest.
- **Interactive Tooltips**: Built-in definitions for complex financial jargon (SIP, HRA, 80C, Standard Deduction) hidden behind intuitive hover icons for a clean aesthetic.
- **FIRE Path Planner & Goal Masterplan**: Calculate systematic investment strategies to conquer milestones like buying a house, saving for retirement, or building an emergency fund.
- **Dark Mode UI**: State-of-the-art visual aesthetics using Shadcn components and Tailwind CSS `slate-ocean` gradients.

## 💻 Tech Stack
- **Frontend Framework**: React 19 + Vite
- **Routing**: React Router DOM v7
- **Styling**: Tailwind CSS v3
- **Components/UI**: Radix UI + Custom Glassmorphism patterns
- **Data Visualization**: Recharts (Radar / Ring Charts)
- **Icons**: Lucide React
- **Persistence**: Browser `localStorage` (serverless MVP for the hackathon demo)

## ⚡ Getting Started
1. Clone the repository and run `npm install`
2. Run `npm run dev` to start the local Vite server.
3. Access the web app at `http://localhost:5173`
4. Use the built-in **Demo Account** (`demo@moneym.ai`) via the 'Try Demo Account' button for instant access.

## 🎨 Theme Specs

- Surface Layers: Transparent slate with high-contrast borders
- Accents: Voids of `sky-500` & `indigo-500`
